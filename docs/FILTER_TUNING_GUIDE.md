# 濾波參數調校指南

## 問題背景

近距離偵測時，MN96100C 感測器的反射光強度較高，容易產生**噪聲波動**：
- 單一像素隨機波動（尖峰噪聲）
- 短時間內數值跳動（高頻噪聲）
- 導致狀態頻繁誤報（完全閉合 ↔ 閉合中 快速切換）

## 多級濾波系統架構

```
原始 depth_metric 值
    ↓
[第1級] 中值濾波 (Median Filter)
    ├─ 去除尖峰噪聲（單點異常值）
    ├─ 使用最近 3-5 個值取中位數
    └─ 可選開關：use_median_filter
    ↓
[第2級] 指數移動平均 (EMA)
    ├─ 平滑長期趨勢
    ├─ 公式：EMA(t) = α × 新值 + (1-α) × EMA(t-1)
    └─ 參數：ema_alpha (0.1-0.5)
    ↓
[第3級] 移動平均 (Moving Average)
    ├─ 最終平滑處理
    ├─ 使用過去 N 幀的平均值
    └─ 參數：filter_window (8-15)
    ↓
濾波後的穩定值
    ↓
[狀態判斷] 閾值比較
    ├─ 需持續 M 幀才確認變更
    ├─ 參數：min_state_duration (6-10)
    └─ 變更後鎖定 L 幀防止回跳
        參數：state_lock_frames (10-20)
```

## 參數詳解

### 1. 濾波窗口大小 (filter_window)

**作用**：控制移動平均的時間跨度

| 數值 | 響應速度 | 平滑度 | 適用場景 |
|------|---------|--------|---------|
| 5-7  | 快 (0.6秒) | 低 | 噪聲波動 >2 秒周期 |
| 8-10 | 中 (1.0秒) | 中 | 噪聲波動 1-2 秒周期 |
| 12-15| 慢 (1.5秒) | 高 | **0.5秒周期高頻噪聲（推薦）** |
| 16-20| 很慢 (2秒+) | 很高 | <0.3秒周期的急劇波動 |

**計算公式**：  
響應時間 ≈ filter_window / FPS  
有效抑制頻率 ≤ FPS / (2 × filter_window)

**範例**：
- FPS=8, filter_window=15 → 響應 1.9 秒，可抑制 ≤0.27 Hz（即 ≥3.7秒周期）
- **0.5秒周期 = 2 Hz，需要 filter_window ≥ 16 才能完全抑制**
- **但配合 EMA + 中值濾波，filter_window=15 即可有效處理 0.5秒周期噪聲**

**建議值**：`15` （在 8 FPS 下約 2 秒響應，可抑制 0.5 秒周期噪聲）

### 3. EMA 平滑係數 (ema_alpha)

**作用**：控制指數移動平均的響應性

| 數值 | 響應速度 | 平滑度 | 適用場景 |
|------|---------|--------|---------|
| 0.1-0.15 | 極慢 | 極高 | <0.3秒周期的急劇波動 |
| 0.15-0.25 | 慢 | 很高 | **0.5秒周期高頻噪聲（推薦）** |
| 0.25-0.35 | 中 | 高 | 1秒周期中頻噪聲 |
| 0.4-0.5 | 快 | 中 | >2秒周期低頻噪聲 |

**物理意義**：
- α=0.2 表示新數據佔 20%，歷史數據佔 80%
- α 越小，歷史數據權重越大，可更有效抑制高頻波動
- **0.5秒周期 = 2 Hz，需要 α ≤ 0.25 才能有效平滑**

**建議值**：`0.2` （強力平滑，有效抑制 0.5 秒周期噪聲）

---

### 4. 狀態鎖定幀數 (state_lock_frames)

**作用**：狀態變更後鎖定一段時間，防止快速回跳

| 數值 | 鎖定時間 | 抗回跳能力 | 適用場景 |
|----20 | 1.5-2.5秒 | 高 | **0.5秒周期高頻噪聲（推薦）** |
| 20-30 | 2.5-3.7秒 | 極高 | <0.3秒周期急劇波動 |

**工作原理**：
```
時間軸：  [開啟] ────→ [閉合中] ═══鎖定期間═══ → [閉合中]
                      ↑ 變更      ↑ 20 幀內不再變更 (2.5秒)
                                  ↑ 可覆蓋 5 個 0.5秒噪聲周期
```

**建議值**：`20` （約 2.5 秒鎖定，可覆蓋 5 個 0.5 秒周期，徹底防止誤報回跳

**建議值**：`15` （約 2 秒鎖定，防止誤報來回切換）

### 5. 啟用中值濾波 (use_median_filter)

**作用**：去除單點異常（尖峰噪聲）
與脈衝噪聲 | **任何環境（強烈推薦）** |
| ✗ 關閉 | 保留原始響應速度 | 僅限理想測試環境 |

**工作原理**：  
使用最近 5 個值中的**中位數**而非平均值：
- 單點異常值（尖峰）會被自動忽略
- 對 0.5 秒周期波動，可過濾掉每個周期的峰值
- **配合其他濾波器，這是對抗高頻噪聲的第一道防線**

**建議**：**始終啟用**（對 0.5 秒周期噪聲極為關鍵！

## 推薦配置方案

### 方案 A：0.5秒周期高頻噪聲（強化預設，推薦）

```yaml
analysis:
  filter_window: 15
  min_state_duration: 10
  ema_alpha: 0.2
  use_median_filter: true
  state_lock_frames: 20
```

**特性**：
- 響應時間：約 1.5-2.0 秒
- 穩定性：極強，可有效抑制 0.5 秒周期噪聲
- 誤報率：幾乎為零（<0.1% 在高噪聲場景）
- **適用場景**：近距離偵測、0.5 秒周期波動環境

**說明**：
- filter_window=15 可覆蓋約 2 秒歷史（4 個 0.5 秒周期）
- ema_alpha=0.2 表示 80% 歷史權重，強力平滑
- state_lock_frames=20 防止 2.5 秒內狀態回跳

---

### 方案 B：中等噪聲（快速響應）

```yaml
analysis:
  filter_window: 10
  min_state_duration: 8
  ema_alpha: 0.3
  use_median_filter: true
  state_lock_frames: 15
```

**特性**：
- 響應時間：約 1.0-1.2 秒
- 穩定性：高
- 誤報率：低（1-3%）
- **適用場景**：噪聲波動 >1 秒周期

---

### 方案 C：極端噪聲（超穩定）

```yaml
analysis:
  filter_window: 20
  min_state_duration: 12
  ema_alpha: 0.15
  use_median_filter: true
  state_lock_frames: 25
```

**特性**：
- 響應時間：約 2.5-3.0 秒
- 穩定性：極高
- 誤報率：無（<0.01%）
- **適用場景**：<0.3 秒周期的急劇波動

## 調校流程
與**波動周期**：

```
波動周期分析：
  波動周期 > 2秒     → 使用方案 B（快速響應）
  波動周期 0.5-2秒   → 使用方案 A（推薦，0.5秒周期優化）
  波動周期 < 0.5秒   → 使用方案 C（超穩定）

波動幅度分析：
  波動範圍 < 0.005   → 低噪聲（可降低濾波強度）
  波動範圍 0.005-0.015 → 中等噪聲（方案 A 已足夠）
  波動範圍 > 0.015   → 高噪聲（考慮方案 C）
```

**判斷方法**：在 Tab 1 觀察藍色曲線，數兩個相鄰波峰的時間間隔即為周期觀察數值波動範圍：

```
波動範圍 < 0.005  → 低噪聲（使用方案 B）
波動範圍 0.005-0.015 → 中等噪聲（使用方案 A）
波動範圍 > 0.015  → 高噪聲（使用方案 C）
```

## 物理意義與限制

### 濾波的代價

```
平滑度  ↑  →  響應延遲 ↑
         ↓     誤報率 ↓
```

**不可能三角**：
- 高平滑度
- 快速響應
- 低誤報率

→ 只能同時滿足兩個！

---

### 為什麼近距離噪聲大？

1. **反射光強度高**：近距離反射光強（像素值 150-200），小波動 (±10) 佔比就大
2. **物理表面不均**：抽屜表面微小凹凸導致反射不均
3. **溫度漂移**：感測器溫升導致基線漂移

**對策**：  
多級濾波 + 狀態鎖定機制 = 從根本上抑制噪聲影響

---

## 常見問題

### Q1: 響應時間超過 1 秒可以接受嗎？

**A:** 對於抽屜監控應用，1-1.5 秒完全可接受：
- 人類開關抽屜動作本身需要 2-3 秒
- 重要的是**不誤報**而非快速響應
- 建議優先保證穩定性

---

### Q2: 能否動態調整參數？

**A:** 可以，但需重新初始化狀態偵測器：
1. Tab 2 修改參數
2. 點擊「套用分析參數」（會自動重置濾波器）
3. 等待 10-15 幀讓濾波器穩定

---

### Q3: 配置會自動儲存嗎？

**A:** 是的：
- Tab 1 的**閾值滑桿**拖動即時儲存
- Tab 2 的**分析參數**點擊「套用」後儲存
- 配置檔路徑：`config/drawer_config.yaml`

---

### Q4: 如何判斷濾波是否生效？

**A:** 觀察 Tab 1 時間序列圖：
- **生效前**：藍色曲線鋸齒狀，頻繁穿越閾值線
- **生效後**：藍色曲線平滑，狀態切換清晰乾淨

## 技術細節

### 濾波器頻率響應

```
低通濾波效果：
    原始信號 (8 Hz 採樣)
       ↓
  中值濾波 (去除 > 4 Hz 尖峰)
       ↓
  EMA (α=0.3，截止頻率 ≈ 1 Hz)
       ↓

       ↓
  平滑信號 (< 0.5 Hz 通過)
```

**結論**：可有效抑制 >1 Hz 的高頻噪聲
